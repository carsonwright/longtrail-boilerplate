# Longtrail-boilerplate
## Creating your App
```
git clone git@github.com:carsonwright/longtrail-boilerplate.git your-app
```

## Migrations
### Forward
```
yarn db:migrate
```
### Backward
```
yarn db:rollback
```

## Running
### development mode
```
yarn dev
```

### production mode
```
yarn start
```
